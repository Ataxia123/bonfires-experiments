DAOs have a memory problem. Decisions live in Discord threads. Governance context is scattered across Snapshot proposals, forum posts, and Telegram chats. New members join with zero context. Experienced members leave and take institutional knowledge with them.

A Bonfire gives your DAO a shared brain.

## What It Does

**Captures institutional memory.** Every conversation the agent witnesses — governance debates, working group discussions, casual strategy talk — feeds the knowledge graph. Decisions, rationale, and context are preserved automatically.

**Answers governance questions.** "Why did we vote against Proposal 47?" "What was the treasury allocation debate about last month?" "Who's been most active in the governance working group?" The agent draws on months of accumulated context.

**Surfaces connections.** The knowledge graph reveals relationships between people, projects, proposals, and ideas that no single member could track. Who's working on overlapping problems? Which proposals address the same underlying issue?

**Supports decision-making.** By maintaining a structured record of past decisions and their outcomes, the agent provides context for future governance — reducing repeated debates and helping the DAO learn from its own history.

## What This Looks Like

1. **Add the agent to your DAO's Telegram or Discord** — governance channels, working group chats, general discussion
2. **The agent listens and learns** — every 20 minutes, conversations are processed into the knowledge graph
3. **Members query naturally** — tag the agent with questions, get answers grounded in your DAO's actual history
4. **Knowledge compounds** — the longer it runs, the deeper the institutional memory

## Case Study: JoshDAO

JoshDAO (137 members, Telegram-based) deployed a Bonfire as both community infrastructure and a proof point for identity-based DAOs. The agent captures collective knowledge, tracks governance decisions, and serves as the group's queryable institutional memory — while demonstrating that even meme-origin communities can build real coordination value.
